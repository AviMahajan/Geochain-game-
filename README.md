🌍 GeoChain

GeoChain is a web-based geography puzzle game where players connect a Start country to an End country by guessing intermediate countries that share real-world land borders.

It’s designed to make geography intuitive, interactive, and a bit challenging — without relying on maps or hints.

🎮 How It Works
Each level gives you:
A Start country
An End country
Your goal:
Guess the countries that connect them through valid land borders
Game mechanics:
✅ Correct guesses are added to the path
❌ Wrong guesses are tracked separately
🔢 Limited attempts per level (5 attempts)
🏁 Reach the end country before running out of attempts to win
🚀 Features
Clean and minimal UI
Smart country suggestions (prevents typos & improves UX)
Progressive difficulty across levels
Instant feedback on each guess
Designed for quick, engaging gameplay loops
🧠 Design Thinking

GeoChain was built with a focus on:

Tight feedback loops → Guess → feedback → retry
Low friction UX → No spelling errors, smooth input
Implicit learning → Learn geography by playing, not memorizing
Balanced difficulty → Easy onboarding → progressively harder levels
🛠 Tech Stack
Frontend: React + Vite
Styling: Tailwind CSS
Maps: react-simple-maps
Deployment: Vercel
⚙️ Setup & Run Locally
# Clone the repo
git clone https://github.com/your-username/geochain.git

# Navigate into the project
cd geochain

# Install dependencies
npm install

# Run the app
npm run dev
